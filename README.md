# Next.js Kanban App

You need the database to run the program locally

```
docker-compose up -d
```

- -d for run it **detached**

- MongoDB Local URL:

```
mongodb://localhost:27017
```

## Setup env variables

Rename **.env.template** as **.env**

## Fill DB with example data

```
http://localhost:3000/api/seed
```
